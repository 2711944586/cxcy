## 3. 正文引用原则

1. 政策来源用于证明方向和合规需求，不直接夸大为 SkyGuard 已获得政策背书。
2. 本地数据用于支撑图表和趋势分析，成稿时必须说明字段口径、时间范围和清洗方法。
3. 图片资料只做场景、政策、行业案例或 Demo 截图佐证，不作为市场规模数据来源。
4. 所有市场规模、财务预测和客户数量必须在正文、图表、财务表中保持一致。
5. 涉及真实部署、真实客户、真实传感器接入时，如暂无证据，只能写“计划、拟、下一阶段、试点目标”。
